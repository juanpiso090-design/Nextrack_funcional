# 💻 Documentación de Ambientes del Proyecto - Nextrack

## Ambiente de Desarrollo
* **Arquitectura:** Cliente-Servidor Desacoplada (Frontend/Backend).
* **IDE Recomendado:** Visual Studio Code.
* **Tecnologías Frontend:** React JS v18+, Vite (Manejador de entornos), HTML5, JavaScript Moderno.
* **Tecnologías Backend:** Node.js v20, Express JS Framework.
* **Mecanismos de Seguridad:** Middleware CORS habilitado para comunicación inter-puertos segura.

## Ambiente de Pruebas
* **Servidor Local:** Ejecutado sobre entorno de Node mediante comandos `npm start`.
* **Pruebas Unitarias:** Implementadas con scripts nativos de validación de lógica condicional en `test.js`.
* **Pruebas de Integración:** Cliente HTTP (Thunder Client) para verificar el envío de cabeceras y payloads JSON.
